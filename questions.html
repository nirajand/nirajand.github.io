Questions For Web Development II - Final Term




Very Short Answer Questions (2 marks each):

  1. What is the difference between Web 1.0, Web 2.0, and Web 3.0?
  2. Define asynchronous programming in the context of web development.
  3. What is the purpose of an XML namespace?
  4. What is the difference between let and const in JavaScript?
  5. Explain the role of digital certificates in web security.
  6. What is the MVC (Model-View-Controller) architecture?
  7. What is the role of CSS in web development?
  8. Explain the concept of cross-site scripting (XSS) attacks.
  9. What is the purpose of the MVC (Model-View-Controller) architecture in web application development?
  10. What is the purpose of media queries in CSS?
  11. What is the difference between Web 1.0 and Web 2.0?

Short Answer Questions (5 marks each):

  1. Describe the process of handling HTTP requests and responses in a web server.
  2. Explain the concept of responsive web design and how it can be achieved using CSS media queries.
  3. Write an XML document representing a simple book catalog, including elements for title, author, publisher, and publication year.
  4. Implement a JavaScript function that uses promises and the fetch API to retrieve data from a JSON endpoint.
  5. Discuss the importance of input validation and sanitization in web security, and provide examples of techniques for mitigating common vulnerabilities like SQL injection and XSS.
  6. Compare and contrast Bootstrap and CSS Grid/Flexbox for creating responsive layouts.
  7. Explain the concept of unit testing in web application development and its importance.
  8. Write a JavaScript function that uses promises and the async/await syntax to fetch data from an API.
  9. Describe the concept of digital certificates and their role in web security.
  10. Create an XML document that represents a simple data structure, such as a book catalog, and include at least one XML namespace and one XML attribute.
  11. Explain the advantages and disadvantages of using a CSS framework like Bootstrap for web development.
  12. Explain the concept of encryption and decryption in web security, and discuss the role of digital certificates and secure sockets layer (SSL) in ensuring secure communication.

Long Answer Questions (7.5 marks each):

  1. Design and implement a responsive web page layout using CSS Grid or Flexbox, with appropriate styling and media queries for different screen sizes. Explain the advantages of your chosen layout technique over traditional methods.
  2. Develop a JavaScript class that represents a shopping cart for an e-commerce website. The class should have methods for adding items, removing items, calculating the total cost, and applying discounts. Implement inheritance to create a subclass for a "premium" shopping cart with additional features.
  3. Describe the process of securing a web application, from the database design and server-side scripting to client-side validation and authentication. Discuss various security measures and best practices, such as encryption, salting and hashing of passwords, and protection against common web application attacks like XSS and CSRF.
  4. Create an XML document that represents a simple data structure, such as a book catalog or a student record system. Validate the XML document using either a Document Type Definition (DTD) or an XML Schema Definition (XSD). Explain the process and the importance of validating XML documents.
  5. Implement a class in JavaScript that represents a simple e-commerce shopping cart. The class should have methods for adding items to the cart, removing items from the cart, and calculating the total cost of the items in the cart. Demonstrate the use of inheritance by creating a subclass that extends the shopping cart class with additional functionality, such as applying discounts or handling shipping costs.
  6. Discuss the principles of web security and the potential vulnerabilities that web applications may face. Explain the concepts of authentication, encryption, and input validation, and provide examples of how these can be implemented in a web application to mitigate security risks. Additionally, describe strategies for securing a web application, such as implementing SSL/TLS and protecting against common attacks like SQL injection and cross-site scripting (XSS).
  7. Implement a simple JavaScript class that represents a bank account, with properties for account number, balance, and account holder name. Include methods for depositing and withdrawing funds, as well as inheritance for creating different types of accounts (e.g., savings, checking). Demonstrate the use of constructor functions, getters, and setters.
  8. Design and implement a responsive web page layout using CSS Grid or Flexbox. The layout should include a header, navigation menu, main content area, and footer. Explain the advantages of using these modern layout techniques over traditional methods, such as floats and positioning. Incorporate media queries to adjust the layout for different screen sizes and device types.
